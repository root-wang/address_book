# 通讯录

## 系统需求

1. ### 添加联系人

   通过输入姓名和电话自动加入到结构体的数组当中，结构体具有自动计数功能无需设置序号，每次添加成功后通讯录的条数

   ```c++
   list1.num++;
   ```

   自动加1，这个数据可以做添加联系人的序号数，也可以做通讯录的总人数，后面如果需要进行逻辑清空则需要对该数据置零。

   ```c++
   list1.num=0;
   ```
